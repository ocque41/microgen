Docs for Web Scrapers & AI Agents

This page makes automated access predictable, respectful, and useful. Human operators are welcome too.

Machine-readable index

Preferred discovery: /docs/ai.json (JSON). It lists endpoints, fields, and examples.

Access etiquette

  • Respect /robots.txt. We explicitly define allowances for common AI crawlers including GPTBot.
  • Rate: up to 1 request per second; back off for 429 or 503 responses.
  • Send headers: Accept: application/json for JSON; If-Modified-Since to save bandwidth.

Important notes

  • Alt text follows W3C guidance; treat provided descriptions as authoritative when present.
  • Structured data uses JSON-LD only, aligning with Google’s recommendation and omitting deprecated SearchAction markup.
